Description and History

Introduction

Project Kotlin was initially unveiled by JetBrains in 2011 after development for under a year.

  • Written as a new language for the JVM
  • Wanted to create a language that compiled as fast as Java with the featureset of Scala
  • Open sourced in 2012 under the Apache 2 license

Kotlin is object oriented. As it is written for the JVM, Kotlin is completely interoperable with any Java code. This makes it easier than ever for companies to transition their existing Java codebases to Kotlin.

Timeline

  • February 15, 2016 - Kotlin 1.0 released
  • May 17, 2017 - Google announces full support for Kotlin on Android at Google I/O 2017
  • November 28, 2017 - Kotlin 1.2 released with support for sharing code between JVM and JavaScript platforms
  • October 29, 2018 - Kotlin 1.3 released with support for asynchronous programming
  • May 7, 2019 - Google announces Kotlin is its preferred language for Android development

This is an example of the when conditional in kotlin

/* Andrew Moorcroft
* This is a simple example of when block in kotlin.
* This program tries to find the minimum amoung three numbers.
fun main(args: array) {
 var x = 3
 var y = -1
 var z = 10
 when {
  (w <= x && w <= y && w <=z) -> println("w is the minimum")
  (x <= w && x <= y && x <=z) -> println("x is the minimum")
  (y <= w && y <= x && y <= z) -> println("y is the minimum")
  else -> println("z is the minimum")
 }
}
